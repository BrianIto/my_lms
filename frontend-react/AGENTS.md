# Frontend Agent Notes

This app was scaffolded with TanStack CLI using React, shadcn/ui, TanStack Query, TanStack Form, Storybook, Biome, Nitro, Store, and MCP add-ons.

## Common commands

- `bun --bun run dev` - start local dev server
- `bun run build` - build app
- `bun run check` - run checks if available

Prefer existing TanStack Router, Query, Form, Store, and shadcn patterns already generated in this project.
